# ARAD-Auto-Launcher
ARAD Auto Launcher

##動作に必要なもの
1. NeopleCustomURLBuilder
 
   既にChromeやFireFoxを利用している方はインストール済みとなっているはずです。

   [http://webdown2.nexon.co.jp/arad/real/plugin/NeopleCustomURLBuilder.exe](http://webdown2.nexon.co.jp/arad/real/plugin/NeopleCustomURLBuilder.exe)

2. .NET Framework 4.0

   [https://www.microsoft.com/ja-jp/download/details.aspx?id=17851](https://www.microsoft.com/ja-jp/download/details.aspx?id=17851)
